# Advocate Diary — Pro UI Upgrade Pack

Adds:
- Advanced glass/gradient theme, soft shadows
- Sticky Topbar with **Command Palette (Ctrl/⌘ K)**
- Sidebar with active styling
- Cleaner Dashboard cards & chart
- Theme toggle

## Apply
1) Copy files into your frontend, merge `package.json` deps.
2) `npm install`
3) Use the new layout in your router:
```jsx
import Layout from './layouts/Layout.jsx'
// ...
```
4) `npm run dev`

Hotkeys: **Ctrl/⌘ + K** for the Command Palette.
